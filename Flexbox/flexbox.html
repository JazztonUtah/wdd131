A Media Card Example

Media Card Title

This is a media card component that uses Flexbox to control the format of blocks of content consisting of text and images. The behavior we want is for the image to be to the side of the text when there is enough room on wider screens, and the image to stack above the text on narrower screens.

A Gallery Example

Gallery Card 1

This is a media card component in a gallery layout. Flexbox allows us to create responsive galleries that stack on small screens and tile on wider screens.

Gallery Card 2

These cards will automatically adjust their layout based on the available space, creating a flexible and responsive design.
